# Pack widget d'installation — HOP

## Fichiers inclus

- `index.html` : page d’installation
- `manifest.json` : configuration du raccourci écran d’accueil
- `A_REMPLACER_PAR_TON_LOGO.jpg.txt` : mémo à supprimer après ajout du logo

## À faire

1. Ajouter ton logo dans le dossier.
2. Le renommer exactement : `logo.jpg`
3. Dans `manifest.json`, remplacer :
   `https://REMPLACER-PAR-LE-LIEN-DE-L-APPLICATION-HOP`
   par le vrai lien de l’application HOP.
4. Dans `index.html`, remplacer le même lien dans le bouton.

## Dossier final attendu

```txt
index.html
manifest.json
logo.jpg
```

## Utilisation

1. Envoyer le lien de cette page ou un QR code.
2. La personne ouvre la page.
3. Elle ajoute à l’écran d’accueil.
4. L’icône HOP reste sur le téléphone.
